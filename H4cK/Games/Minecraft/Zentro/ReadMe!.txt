📦 Requirements
Minecraft Java Edition
NeoForge 1.21.1
Zentro Client JAR file: Zentro-neoforge-1.21.jar
ZentroMC Launcher (ZentroMC.exe)
A working internet connection
🛠 Installation Guide
1. Install NeoForge 1.21.1
Download and install NeoForge 1.21.1
Run the installer
Make sure the NeoForge profile appears in your Minecraft Launcher
2. Install Zentro Client
Open your .minecraft folder
Windows: Win + R → type %appdata%\.minecraft
Open the mods folder
If it does not exist, create it manually

Place the file:

Zentro-neoforge-1.21.jar

inside the mods folder

3. Start ZentroMC

Download and run:

ZentroMC.exe
If not installed yet, let it complete setup automatically
Launch Minecraft through NeoForge profile
🎮 In-Game Usage
When Minecraft starts, the Zentro interface will appear automatically
A module window will open showing all available features
The client will operate as an overlay on top of Minecraft
Window Behavior
Minecraft may start in windowed mode
Zentro will manage rendering and overlay systems
The client interface will stay visible above the game window
Fullscreen behavior may be adjusted automatically for stability
